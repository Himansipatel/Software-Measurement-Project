Build Plugin

This is a Maven 2.x/3.x Ant Plugin which is used by Apache Commons builds. See the Development page for information to help maintain this plugin.

Plugin Goals

Available Goals:

  • commons:all - Generate all pages for the component's latest release (but not sandbox pages.)
  • commons:all-sandbox - Generate all pages for the sandbox component's latest release.
  • commons:download-page - Generate a Download page for the component's latest release.
  • commons:mail-page - Generate a custom Mailing List page for a component.
  • commons:jira-page - Generate a custom JIRA Issue Tracking page for a component.
  • commons:sandbox-jira-page - Generate a custom JIRA Issue Tracking page for a Sandbox component.
  • commons:copy-javadoc-files - Copy NOTICE and LICENSE files to -javadoc jar.
  • commons:readme-md - Generate a README.md file containing information for contributors from GitHub.
  • commons:contributing-md - Generate a CONTRIBUTING.md file containing information on how to contribute for contributors from GitHub.

Trying out the plugin

Configure the plugin in the <build> section of the pom.xml 

<plugin>
  <groupId>org.apache.commons</groupId>
  <artifactId>commons-build-plugin</artifactId>
  <version>1.9</version>
</plugin>

Configure any properties required by the goal and try executing.
